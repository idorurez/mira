# Live2D Character Commission Brief

## Project Overview

I need a character illustration prepared for Live2D Cubism rigging. I have reference images showing the character design and style — the artist should match this closely.

**Deliverable:** Layered PSD file, ready for Live2D rigging (I will handle rigging myself)

---

## Technical Requirements

### Canvas Size
- **2048x2048 minimum** (4096x4096 preferred for detail)
- Character centered, upper body focus (head to mid-torso)
- Transparent background

### Layer Organization
All parts must be on **separate named layers** for deformation rigging. Parts should extend slightly beyond their visible edges (for movement without gaps).

---

## Required Layers

### Face
- `face_base` — skin only, no features, extends under hair/eyes
- `nose` — separate if 3D-style, can be part of face_base if flat anime style

### Eyes (Left and Right separate — 14 layers total)
- `eye_white_L` / `eye_white_R`
- `iris_L` / `iris_R`
- `pupil_L` / `pupil_R`
- `eyelid_upper_L` / `eyelid_upper_R`
- `eyelid_lower_L` / `eyelid_lower_R`
- `eyelash_upper_L` / `eyelash_upper_R`
- `eyelash_lower_L` / `eyelash_lower_R` (if character has visible lower lashes)

### Eyebrows (2 layers)
- `eyebrow_L`
- `eyebrow_R`

### Mouth (4+ layers)
- `upper_lip`
- `lower_lip`
- `mouth_inside` — dark interior, tongue (visible when mouth opens)
- `teeth_upper` (optional, if character shows teeth when smiling)
- `teeth_lower` (optional)

### Hair (4+ layers, for physics simulation)
- `hair_back` — hair behind head/shoulders
- `hair_front` — bangs, front pieces
- `hair_side_L` — strands on left side
- `hair_side_R` — strands on right side
- Additional strand layers welcome for better physics

### Body (2-3 layers)
- `neck`
- `torso` / `body`
- `collar` or clothing details (if separate from body)

### Effects/Extras
- `eye_glow` — glowing effect around eyes (separate overlay, additive blend)
- `face_markings` — any tattoos/markings as separate layer
- `blush` (optional overlay for expressions)

---

## Character Design

### Style
- Anime style
- Cyberpunk aesthetic
- Clean lines, detailed but not overly complex

### Appearance
- **Hair:** White bob-cut, slight purple/blue tint, clean strands
- **Eyes:** Large expressive anime eyes, purple/pink irises, glowing effect
- **Face:** Soft features, face markings under eyes (tech-style lines)
- **Expression:** Neutral for base — relaxed mouth, eyes open, calm brows
- **Outfit:** White bodysuit/exosuit, visible collar/neckline, subtle tech details
- **Pose:** Facing forward, very slight 3/4 angle okay, relaxed shoulders

### Reference Images
I will provide:
- 10+ expression reference images (512x512)
- Full body A-pose reference
- Close-up face reference
